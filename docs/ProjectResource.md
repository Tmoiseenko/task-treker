# ProjectResource - MoonShine Ресурс для управления проектами

## Обзор

ProjectResource предоставляет полнофункциональный интерфейс для управления проектами в административной панели MoonShine.

## Возможности

### Основные поля

- **ID** - уникальный идентификатор проекта
- **Название** - название проекта (обязательное)
- **Описание** - подробное описание проекта (опциональное)
- **Тип** - тип проекта (обязательное):
  - Мобильное приложение (mobile_app)
  - Telegram бот (telegram_bot)
  - Веб-сайт (website)
  - CRM система (crm_system)
- **Статус** - текущий статус проекта (обязательное):
  - Активный (active)
  - Завершенный (completed)
  - Архивный (archived)
  - На паузе (on_hold)

### Связи many-to-many

#### Этапы (stages)
Проект может иметь несколько этапов работы. При создании проекта система автоматически предлагает этапы на основе выбранного типа:

- **Мобильное приложение**: Дизайн, Бэкенд, Мобилка, Тестирование
- **Telegram бот**: Дизайн, Бэкенд, Тестирование
- **Веб-сайт**: Дизайн, Бэкенд, Админка, Фронтенд, Тестирование
- **CRM система**: Дизайн, Бэкенд, Админка, Фронтенд, Тестирование

#### Участники (members)
Проект может иметь несколько участников (пользователей). Участники получают доступ к задачам проекта.

## Использование

### Создание проекта

1. Перейдите в раздел "Проекты" в административной панели
2. Нажмите "Создать"
3. Заполните обязательные поля: название, тип, статус
4. Выберите этапы (система предложит этапы на основе типа)
5. Добавьте участников проекта
6. Сохраните

### Редактирование проекта

1. Откройте проект из списка
2. Измените необходимые поля
3. Добавьте или удалите этапы
4. Добавьте или удалите участников
5. Сохраните изменения

### Фильтрация

В списке проектов доступны фильтры:
- По названию
- По типу проекта
- По статусу

### Поиск

Поиск работает по следующим полям:
- ID проекта
- Название
- Описание

## Технические детали

### Расположение файлов

```
app/MoonShine/Resources/Project/
├── ProjectResource.php          # Основной ресурс
└── Pages/
    ├── ProjectIndexPage.php     # Страница списка
    └── ProjectFormPage.php      # Страница формы
```

### Регистрация

Ресурс зарегистрирован в `app/Providers/MoonShineServiceProvider.php`

### Группировка в меню

Ресурс находится в группе "Управление проектами" с иконкой портфеля (briefcase).

## Связанные требования

- Requirements 2.1: Создание проекта с названием, описанием и типом
- Requirements 2.5: Применение шаблона этапов для типа проекта
- Requirements 2.11: Выбор этапов из справочника
- Requirements 2.12: Удаление этапов из проекта

## Тестирование

Функциональность покрыта тестами:
- `tests/Feature/ProjectResourceTest.php` - базовые тесты ресурса
- `tests/Feature/ProjectManagementTest.php` - интеграционные тесты управления проектами

Запуск тестов:
```bash
./vendor/bin/sail artisan test --filter=Project
```
