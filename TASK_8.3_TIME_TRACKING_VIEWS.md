# Задача 8.3: Blade представления для учета времени

## Обзор

Реализованы Blade представления для полноценного учета времени в системе управления проектами.

## Созданные компоненты

### 1. Компонент учета времени по этапу задачи
**Файл:** `resources/views/tasks/partials/time-tracking.blade.php`

**Функциональность:**
- ✅ Таймер для трекинга времени (Alpine.js)
  - Запуск/остановка таймера
  - Отображение времени в реальном времени (HH:MM:SS)
  - Автоматическая проверка статуса каждые 5 секунд
  - Сохранение времени при остановке
  
- ✅ Форма добавления оценки времени
  - Ввод оценки в часах (0.1 - 1000)
  - Обновление существующей оценки
  - Отображение всех оценок от разных пользователей
  
- ✅ Форма добавления записи времени вручную
  - Ввод часов (0.1 - 24)
  - Выбор даты (не позднее сегодня)
  - Опциональное описание работы
  - Автоматический расчет стоимости
  
- ✅ Список записей времени
  - Отображение всех записей по этапу
  - Информация о пользователе, дате, часах и стоимости
  - Итоговая сумма по этапу

**Использование:**
```blade
@include('tasks.partials.time-tracking', ['taskStage' => $taskStage])
```

### 2. Виджет суммарного времени по задаче
**Файл:** `resources/views/tasks/partials/time-summary.blade.php`

**Функциональность:**
- ✅ Общее время и стоимость по задаче
- ✅ Разбивка по этапам с прогресс-барами
  - Сравнение фактического времени с оценкой
  - Цветовая индикация (зеленый < 80%, желтый 80-100%, красный > 100%)
- ✅ Разбивка по исполнителям
- ✅ Кнопка для раскрытия детального учета времени

**Использование:**
```blade
@include('tasks.partials.time-summary', [
    'task' => $task,
    'totalHours' => $totalHours,
    'totalCost' => $totalCost
])
```

## Технические детали

### Alpine.js интеграция

**Установлены пакеты:**
```bash
./vendor/bin/sail npm install alpinejs @alpinejs/collapse
```

**Обновлен файл:** `resources/js/app.js`
```javascript
import Alpine from 'alpinejs';
import collapse from '@alpinejs/collapse';

Alpine.plugin(collapse);
window.Alpine = Alpine;
Alpine.start();
```

### Функции таймера (Alpine.js)

**Компонент `timeTracking(taskStageId)`:**
- `init()` - инициализация и периодическая проверка статуса
- `checkTimerStatus()` - проверка статуса таймера через API
- `startTimer()` - запуск таймера
- `stopTimer()` - остановка таймера
- `startDisplayTimer()` - запуск отображения времени
- `stopDisplayTimer()` - остановка отображения
- `updateDisplay()` - обновление отображаемого времени каждую секунду

### API endpoints

Используются существующие маршруты:
- `POST /task-stages/{taskStage}/timer/start` - запуск таймера
- `POST /task-stages/{taskStage}/timer/stop` - остановка таймера
- `GET /task-stages/{taskStage}/timer/status` - статус таймера
- `POST /task-stages/{taskStage}/estimates` - создание/обновление оценки
- `POST /task-stages/{taskStage}/time-entries` - создание записи времени
