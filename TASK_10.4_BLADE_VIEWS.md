# Task 10.4: Blade представления для комментариев и файлов

## Обзор

Реализованы Blade-представления для комментариев, прикрепленных файлов и истории изменений задачи. Все компоненты интегрированы в страницу просмотра задачи.

## Реализованные компоненты

### 1. Комментарии (resources/views/tasks/partials/comments.blade.php)

**Функциональность:**
- Форма добавления комментария с валидацией
- Список комментариев в хронологическом порядке
- Возможность удаления собственных комментариев
- Отображение автора и времени создания
- Скрытие удаленных комментариев (soft delete)
- Placeholder для пустого списка

**Особенности:**
- Textarea с автофокусом и валидацией
- Кнопка "Удалить" только для собственных комментариев
- Визуальное выделение комментариев (border-left)
- Относительное время (diffForHumans)
- Подсчет активных комментариев (без удаленных)

### 2. Прикрепленные файлы (resources/views/tasks/partials/attachments.blade.php)

**Функциональность:**
- Форма загрузки файлов с drag-and-drop интерфейсом
- Список прикрепленных файлов с метаданными
- Кнопка скачивания для всех пользователей
- Кнопка удаления для автора файла, автора задачи и исполнителя
- Отображение размера файла, автора и времени загрузки
- Placeholder для пустого списка

**Особенности:**
- Визуальный индикатор выбранного файла
- Информация о максимальном размере файла
- Иконка файла для визуального представления
- Truncate для длинных имен файлов
- Hover эффекты для улучшения UX

### 3. История изменений (resources/views/tasks/partials/audit-history.blade.php)

**Функциональность:**
- Отображение истории изменений в формате timeline
- Показ автора, времени и типа изменения
- Визуальное сравнение старого и нового значения
- Ограничение до 20 последних записей
- Placeholder для пустой истории

**Особенности:**
- Timeline дизайн с точками и линиями
- Цветовое кодирование (красный для старого, зеленый для нового)
- Абсолютное и относительное время
- Hover эффекты для интерактивности
- Индикатор количества скрытых записей

## Интеграция

Все три компонента интегрированы в `resources/views/tasks/show.blade.php`:

```blade
<!-- Comments -->
@include('tasks.partials.comments', ['task' => $task])

<!-- Attachments -->
@include('tasks.partials.attachments', ['task' => $task])

<!-- Audit Log -->
@include('tasks.partials.audit-history', ['task' => $task])
```

## Маршруты

Используются следующие маршруты:

**Комментарии:**
- `POST /tasks/{task}/comments` - создание комментария
- `DELETE /tasks/{task}/comments/{comment}` - удаление комментария

**Файлы:**
- `POST /tasks/{task}/attachments` - загрузка файла
- `GET /tasks/{task}/attachments/{attachment}/download` - скачивание файла
- `DELETE /tasks/{task}/attachments/{attachment}` - удаление файла

## Стилизация

Все компоненты используют Tailwind CSS для единообразного дизайна:

- Белые карточки с тенью (`bg-white rounded-lg shadow-sm`)
- Отступы и spacing для читаемости
- Цветовая схема: indigo для акцентов, gray для текста
- Hover эффекты для интерактивных элементов
- Responsive дизайн

## Валидация и безопасность

**Комментарии:**
- Валидация через `StoreCommentRequest`
- Проверка авторства при удалении
- Защита от XSS через Blade escaping

**Файлы:**
- Валидация через `StoreAttachmentRequest`
- Проверка размера файла (макс. 10 МБ)
- Проверка прав доступа при скачивании/удалении
- Безопасное хранение в `storage/app/attachments`

**История изменений:**
- Только чтение, изменение невозможно
- Автоматическое логирование через TaskObserver

## Использование

### Добавление комментария
1. Открыть страницу задачи
2. Прокрутить до секции "Комментарии"
3. Ввести текст в textarea
4. Нажать "Отправить"

### Загрузка файла
1. Открыть страницу задачи
2. Прокрутить до секции "Прикрепленные файлы"
3. Выбрать файл через кнопку или drag-and-drop
4. Нажать "Загрузить"

### Просмотр истории
1. Открыть страницу задачи
2. Прокрутить до секции "История изменений"
3. Просмотреть последние 20 изменений
